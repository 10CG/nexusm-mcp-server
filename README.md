# nexusm-mcp-server

Nexusm MCP Server — generic MCP server exposing Nexusm core capabilities (memory, conversation, knowledge, feedback, context) to MCP clients via `@modelcontextprotocol/server` stdio + Streamable HTTP transports.

Published to npm as **`@nexusm/mcp-server`**.
